Фирма "1С", Москва. Октябрь 2014 года

Внешняя обработка «Свертка информационной базы» 

При выполнении свертки происходит формирование документов ввода остатков по 
регистрам на определенную дату (дату свертки) и удаление не используемых документов 
и движений по регистрам (сведений, накопления, бухгалтерии) до даты свертки 
включительно (сворачиваемый период).

Обработка может использоваться для сокращения информационных без перед переходом на
редакцию 3.0

Предназначена для релизов 2.0.62 и выше конфигурации «Бухгалтерия предприятия».

Обработка размещена на пользовательском сайте users.v8.1c.ru на странице конфигурации.

Для свертки информационной базы нужно выполнить следующие действия:
1. Сохранить внешнюю обработку на локальный диск компьютера.
2. Открыть обработку «Свертка информационной базы» (меню «Файл – Открыть»).
3. Следовать инструкциям, описанным в справке к обработке «Свертка информационной базы».
